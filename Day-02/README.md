# Day 2 – PLC Fundamentals

## 🎯 Learning Objective

Understand what a PLC is, why it is used, and its role in industrial automation.

## 📖 What is a PLC?

A Programmable Logic Controller (PLC) is an industrial computer used to automate and control machines and processes.

## 🏭 Where PLCs Are Used

- Manufacturing
- Water Treatment Plants
- Oil & Gas Facilities
- Power Plants
- Building Automation

## ⚙️ Main PLC Components

- CPU
- Power Supply
- Input Modules
- Output Modules
- Communication Modules

## 🔄 PLC Scan Cycle

1. Read Inputs
2. Execute Logic
3. Update Outputs
4. Repeat

## 💡 Key Takeaways

- PLCs are the brains of industrial automation.
- They are designed for reliability in harsh environments.
- PLCs communicate with SCADA systems.
